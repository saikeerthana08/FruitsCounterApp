### Completion Instructions

<details>
<summary>Functionalities added</summary>
<br/>

The app has the following functionalities

- Initially, the count of eaten mangoes and bananas is 0
- When **Eat Mango** is clicked the count of the mangoes eaten is incremented by 1
- When **Eat Banana** is clicked the count of the bananas eaten is incremented by 1

</details>
